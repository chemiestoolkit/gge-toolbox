📐 Castle Distance

The straight-line field distance — the "clicks" — between any two castles. Look each side up by player name (pick which castle), or type raw X:Y coordinates. Great for judging who's in reach.

Pick or enter two castles to measure the distance between them.

Distances are straight-line map fields. Name lookups: gge-tracker.com public API.
