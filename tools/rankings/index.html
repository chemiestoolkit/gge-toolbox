📊 Player & Alliance Rankings

Server rankings for might, glory, loot and honour. Pick a server, switch between players and alliances, and click a column to sort.

About the data: values come from the gge-tracker.com public API, which refreshes per its own crawl — the Updated column shows each row's age. Very high-might accounts can lag behind their live in-game value; for a single player's current figures, a direct profile lookup is more reliable.
