🌊 Storm Islands Rankings

The Storm Islands leaderboard — cargo points (the score) plus the full aquamarine breakdown: total collected, resource isles, storm forts, PvP and spend. Pick a server and page through, or search a player.

About the data: Storm Islands scores and the aquamarine breakdown come from gge-tracker.com's last snapshot (shown in the status line) — the live game protocol doesn't expose the per-source breakdown. Sorted by cargo points.
